# Software Quality Assurance

Greetings!

This repository showcases my software quality assurance practices, the
testing approach and technologies I use, and some works I was able to archive.

------------------------------------------------------------------------

## My Core Testing Approach

- **User-Centric**  
  Prioritizing what matters most to users

- **Risk-Based Testing**  
  Prioritizing what matters most to business/organization

- **Clear Communication**  
  Ensuring everything is documented as straightforward as possible and are actionable
